# 🌸 SmartParenting AI: Jembatan Sains & Hati

Aplikasi asisten pengasuhan cerdas yang menggabungkan kekuatan **Generative AI (Gemini)** dengan **Prinsip Psikologi Parenting** dan **Edukasi Sains (Kimia)**.

## ✨ Fitur Utama
- **💬 Anti-Bentak Transformer:** Mengubah komunikasi reaktif menjadi proaktif & positif.
- **🎨 Katalis Aktivitas:** Ide bermain sensory play berbasis sains untuk regulasi emosi anak.
- **📖 Jurnal Burnout:** Deteksi dini kelelahan mental orang tua dengan analisis sentimen.


- **Framework:** Streamlit (UI ramah pengguna)
- **Engine:** Google Gemini 1.5 Flash API
- **Language:** Python
- **Knowledge Source:** Modul Psikologi Parenting
